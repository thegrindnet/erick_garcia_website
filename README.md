# Erick's Garage Website

A modern, responsive one-page website built with React and CSS for Erick's Garage. This site is designed to showcase automotive repair services and encourage visitors to call for immediate assistance.

## 🚀 Features

- Fully responsive design
- Modern hero section with strong call-to-action
- Services section
- About section
- Contact section
- Click-to-call phone buttons
- Clean and beginner-friendly code
- Built with React and plain CSS
- No TypeScript
- No Tailwind CSS
- No external UI libraries

## 🛠️ Technologies Used

- React
- JavaScript (ES6+)
- JSX
- CSS3
- HTML5
- Vite
